WBJEE Rank Predictor

A static website to predict possible WBJEE colleges using WBJEE 2025 OR-CR cutoff data.

Files:
- index.html: main website
- data.js: cutoff database used by the website
- cutoffs.csv: original cleaned cutoff data

How to run locally:
Open index.html in Chrome/Edge.

How to deploy on Render:
Create Static Site, connect GitHub repo, publish directory: .

Disclaimer: This tool is based on previous year OR-CR only and does not guarantee admission.
